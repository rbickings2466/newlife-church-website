🔐 IMPORTANT: ENVIRONMENT VARIABLES

After deploying, you MUST add these environment variables in Netlify:

  Required for Firebase (Members section & Admin):
    • VITE_FIREBASE_API_KEY
    • VITE_FIREBASE_AUTH_DOMAIN
    • VITE_FIREBASE_PROJECT_ID
    • VITE_FIREBASE_STORAGE_BUCKET
    • VITE_FIREBASE_MESSAGING_SENDER_ID
    • VITE_FIREBASE_APP_ID

  Required for YouTube (Sermons):
    • VITE_YOUTUBE_API_KEY
    • VITE_YOUTUBE_CHANNEL_ID

  Required for Contact Form:
    • VITE_EMAILJS_SERVICE_ID
    • VITE_EMAILJS_TEMPLATE_ID
    • VITE_EMAILJS_PUBLIC_KEY

  Required for AI Chatbot:
    • VITE_GEMINI_API_KEY

  💡 Find these in your .env.local file

🌐 OPTIONAL: SET UP GITHUB FOR AUTO-DEPLOY

  This enables automatic deployment when you make changes:

  1. Create repository on GitHub.com
  2. Run these commands:

     cd "/Users/rbickings/Desktop/New Life Website/newlife-church"
     git remote add origin https://github.com/YOUR-USERNAME/newlife-church-website.git
     git push -u origin main

  3. In Netlify: "New site" → "Import from Git" → Select repo
  4. Every push to GitHub will auto-deploy! 🎉
